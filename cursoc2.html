Printf: Imprimir en pantalla

 

Para utilizar la función printf en nuestros programas debemos incluir la directiva: 

#include <stdio.h>

al principio de programa. Como se ha visto en el programa hola mundo.

Si se quiere imprimir una cadena basta con hacer (no olvides el ";" al final):

printf( "Cadena" );

Esto resultará por pantalla:

Cadena

Lo que se ponga entre las comillas es lo que se muestra por pantalla.

Si se usa otro printf, por ejemplo:

#include <stdio.h>

int main()
{
printf( "Cadena" );
printf( "Segunda" );

return 0;
}

Se obtiene:

CadenaSegunda

Este ejemplo muestra cómo funciona printf. Para escribir en la pantalla se usa un cursor que no se ve...

Si lo que se quiere es sacar cada una en una línea se utiliza "\n".

#include <stdio.h>

int main()
{
printf( "Cadena\n" );
printf( "Segunda" );

return 0;
}

y tendremos:

Cadena
Segunda

También podemos poner más de una cadena dentro del printf:

printf( "Primera cadena" "Segunda cadena" );

Lo que no podemos hacer es meter cosas entre las cadenas:

printf( "Primera cadena" texto en medio "Segunda cadena" );

esto no es válido...

Pero ¿qué pasa si queremos imprimir el símbolo " en pantalla?

Esto es "extraño"

Si hacemos:

printf( "Esto es "extraño"" );

dará error.

La solución es usar \":

printf( "Esto es \"extraño\"" );

La contrabarra sirve para caracteres especiales...

Si queremos imprimir una barra invertida:

printf( "Hola\\Adiós" );

Esto imprime una sola '\'.

Esto no ha sido más que una introducción a printf.
